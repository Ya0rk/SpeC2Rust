# 项目总文档 - MD5-C

## 1. 项目概述

**项目名称**: MD5-C  
**项目描述**: MD5-C 是一个基于 C 语言实现的 MD5 消息摘要算法库。它提供了计算字符串和文件 MD5 哈希值的功能，并且可以通过命令行接口方便地使用。该项目旨在为开发者提供一个简单易用的 MD5 哈希计算工具。

## 2. 项目功能

MD5-C 的主要功能包括：

- **计算字符串的 MD5 哈希值**
- **计算文件的 MD5 哈希值**
- **打印计算得到的 MD5 哈希值**

用户可以通过命令行参数指定要计算的字符串或文件路径，并获取相应的 MD5 哈希值。

## 3. 项目架构

MD5-C 项目采用模块化设计，主要由以下几个模块组成：

- **root 模块**: 根模块负责初始化和协调整个项目。
- **md5.c**: 实现 MD5 算法的核心逻辑。
- **main.c**: 提供命令行接口，调用 MD5 算法模块进行哈希计算。

## 4. 模块关系

### root 模块

- **依赖模块**: `md5.c`
- **功能**: 
  - 解析命令行参数。
  - 调用 `md5.c` 中的函数进行 MD5 哈希计算。
  - 输出计算结果。

### md5.c

- **依赖模块**: 无
- **功能**:
  - 实现 MD5 算法的核心步骤，包括初始化、更新和最终化。
  - 提供 `md5String` 和 `md5File` 函数，分别用于计算字符串和文件的 MD5 哈希值。

### main.c

- **依赖模块**: `md5.c`
- **功能**:
  - 解析命令行参数，判断是要计算字符串还是文件的 MD5 哈希值。
  - 调用 `md5.c` 中的相应函数进行计算。
  - 输出计算结果。

## 5. 技术特点

- **标准实现**: MD5-C 严格遵循 MD5 算法的标准规范，确保计算结果的准确性。
- **高效性**: 使用位操作和循环优化，提高了算法的执行效率。
- **可移植性**: 代码编写简洁，易于在不同平台上编译和运行。
- **易用性**: 通过命令行接口，用户可以方便地进行 MD5 哈希计算。

## 6. 使用说明

### 编译

```sh
gcc -o md5-c main.c md5.c
```

### 运行

#### 计算字符串的 MD5 哈希值

```sh
./md5-c "Hello, World!"
```

#### 计算文件的 MD5 哈希值

```sh
./md5-c /path/to/file
```

### 参数说明

- `-s <string>`: 计算指定字符串的 MD5 哈希值。
- `-f <file_path>`: 计算指定文件的 MD5 哈希值。

## 7. 总结与展望

MD5-C 项目是一个功能完善、性能优良的 MD5 哈希计算工具。通过模块化设计和高效的实现，它满足了大多数用户的使用需求。未来，可以考虑增加对更多哈希算法的支持，以及进一步优化性能，以适应更复杂的应用场景。
